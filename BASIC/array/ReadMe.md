List khác Array ở điểm:
- Array có ép kiểu, kiểm tra kiểu, còn List thì không

```python
a = [1, 3.5, "Hello"]  # Đây là list

import array as arr
a = arr.array('d', [1, 3.5, "Hello"])   # Chuyển sang array lỗi sấp mặt	
```

Để khai báo Array
```python
import array as arr
a = arr.array('d', [1.1, 3.5, 4.5])
print(a)
```
